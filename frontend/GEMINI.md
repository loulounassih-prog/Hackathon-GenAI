This is the `frontend` directory for the HrFlow project. Currently, it is an empty project directory.

## Project Structure

## Getting Started

To initialize this frontend project, you can follow the instructions below based on your preferred technology stack.

### React (Vite)

To create a new React application with Vite, run:

```bash
npm create vite@latest . -- --template react-ts
npm install
```

### Next.js

To create a new Next.js application, run:

```bash
npx create-next-app@latest .
```

## Development Workflow

Once the project is initialized:

1.  **Install dependencies:** Use `npm install` or `yarn install`.
2.  **Start development server:** Use `npm run dev`.
3.  **Build for production:** Use `npm run build`.
4.  **Run tests:** Use `npm test`.
